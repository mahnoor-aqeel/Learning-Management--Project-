# 🎓 NexLearn - Full Stack MERN Learning Management System

A complete, production-ready MERN Stack Learning Management System with role-based authentication, course management, and enrollment tracking.

## 🚀 Features
- **3 User Roles**: Admin, Instructor, Student
- **JWT Authentication** with Bcrypt password hashing
- **Role-Based Access Control** with protected routes
- **Course Management**: Create, Edit, Delete, Publish courses
- **Lesson Management**: Upload lessons per course
- **Student Enrollment** with progress tracking
- **Admin Analytics**: User/Course/Enrollment statistics
- **Responsive Design** with Bootstrap
